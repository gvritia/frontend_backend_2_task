# Система управления товарами (Node.js + React)

Учебный проект, реализующий полноценную систему управления товарами с аутентификацией, JWT-токенами (access + refresh), системой ролей и правами доступа.

## Этапы выполнения

Проект выполнен в соответствии с требованиями практического задания:

- **Этап 1**: Базовое серверное приложение (CRUD товаров + регистрация/логин)
- **Этап 2**: JWT авторизация + защищённые маршруты + `/api/auth/me`
- **Этап 3**: Refresh-токены + маршрут `/api/auth/refresh`
- **Этап 4**: Клиентская часть на React.js
- **Этап 5**: Система ролей (Гость → Пользователь → Продавец → Администратор) + управление пользователями

## Возможности

### Backend (Node.js + Express)
- Регистрация и авторизация пользователей
- Access Token (15 минут) + Refresh Token (7 дней)
- Защита маршрутов и проверка ролей
- CRUD операций с товарами
- Управление пользователями (только для администратора)

### Frontend (React + Vite)
- Страницы регистрации и входа
- Просмотр списка товаров (для всех авторизованных)
- Просмотр детальной информации о товаре
- Добавление и редактирование товаров (Продавец и Администратор)
- Удаление товаров (только Администратор)
- Админ-панель для управления пользователями (блокировка/разблокировка)

## Тестовые аккаунты

| Email                  | Пароль     | Роль       |
|------------------------|------------|------------|
| `admin@example.com`    | `admin123` | admin      |
| `seller@example.com`   | `seller123`| seller     |

После регистрации новый пользователь получает роль **user**.

## Технологии

**Backend:**
- Node.js
- Express
- JWT (access + refresh tokens)
- bcryptjs
- CORS

**Frontend:**
- React.js (Vite)
- React Router DOM
- Axios
- Context API
